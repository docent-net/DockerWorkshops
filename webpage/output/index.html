ABOUT

Workshops will take place on 26th July 2014. We'll cover following topics:
  1. resources management with control - groups
  2. process management with systemd
  3. Ansible automation
  4. creating and managing Docker based Linux containers
  5. use cases for Docker containers

For whom?

Who should attend?

Workshops will be open and free for everyone; keep in mind that there's some entrance level of knowledge you should have in order to be able to fully understand and solve workshops tasks. You should have some basic LAMP system administration experience and good CLI skills.

Keep in mind that we will be using VirtualBox which will help us deploy and build workshop environments.
